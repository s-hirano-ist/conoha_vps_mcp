# 🧰 MCP サーバー対応ツール一覧

ConoHa VPS MCP サーバーに実装されている各ツールと、それぞれが提供する機能一覧をまとめたものです。  
各ツールは OpenStack API をベースに設計されており、サーバー・ボリューム・セキュリティグループなどの主要なリソースを操作可能です。  
それぞれの機能の詳細は、[公開 API ドキュメント](https://doc.conoha.jp/reference/api-vps3/)をご覧ください。

---

## 🔸 ユーティリティツール

### `fetch_url`

指定したURLからコンテンツを取得するツールです。

- URLからテキストコンテンツを取得
- スタートアップスクリプトの取得などに利用

### `encode_base64`

指定した文字列をBase64エンコードするツールです。

- テキストをBase64形式にエンコード
- スタートアップスクリプトのエンコードなどに利用
- 最大10000文字まで対応

---

## 🔹 `conoha_get`

- サーバー詳細一覧取得
- サーバープラン詳細一覧取得
- SSHキーペア一覧取得
- ボリュームタイプ一覧取得
- ボリューム詳細一覧取得
- イメージ一覧取得
- セキュリティグループ一覧取得
- セキュリティグループルール一覧取得
- ポート一覧取得
- スタートアップスクリプト一覧取得
- コンテナ一覧取得
- オブジェクト一覧取得
- オブジェクト詳細取得

---

## 🔹 `conoha_get_by_param`

- サーバーに紐づくIPアドレス一覧取得  
- サーバーに紐づくセキュリティグループ一覧取得  
- サーバー利用状況グラフ取得（CPU使用時間）  
- サーバー利用状況グラフ取得（ディスクIO）  
- セキュリティグループ詳細取得（ID指定）  
- セキュリティグループルール詳細取得（ID指定）  

---

## 🔹 `conoha_post`

- サーバー作成  
- SSHキーペア作成  
- ボリューム作成  
- セキュリティグループ作成  
- セキュリティグループルール作成  

---

## 🔹 `conoha_post_put`

- コンテナ作成
- オブジェクトアップロード

---

## 🔹 `conoha_post_put_by_param`

- サーバー操作  
  - サーバー起動  
  - サーバー停止  
  - サーバー強制停止  
  - サーバー再起動  
  - サーバーリサイズ  
  - サーバーリサイズ承認  
  - サーバーリサイズ取り消し  
- リモートコンソールURL取得  
- ボリュームアタッチ  
- セキュリティグループ更新  
- ポート更新  
- ボリューム更新  

---

## 🔹 `conoha_post_by_header_param`

- アカウント容量設定
- コンテナのWeb公開設定
- コンテナのWeb公開解除

---

## 🔹 `conoha_delete_by_param`

- サーバー削除  
- SSHキーペア削除
- セキュリティグループ削除  
- セキュリティグループルール削除  
- ボリューム削除
- コンテナ削除
- オブジェクト削除

## 🔹 `conoha_head`

- アカウント情報取得
- コンテナ詳細取得
